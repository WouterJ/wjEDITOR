Project opzetten

De code is eigenlijk best simpel. Je moet het editor object eerst aanmaken. Hierbij kun je een paar parameters meegeven: 

var myEditor = editor( input_element[, result_element][, options] );
  • input_element

    Dit is het element waarin de gebruiker zijn tekst bewerkt

  • result_element

    Het element waarin het resultaat moet komen. Dit is alleen nodig als result op true staat in de options.

  • options

    De opties die je kunt invullen de opties zijn:

    • startTag
      - het teken waarmee standaard tags beginnen, default = [
    • endTag
      - het teken waarmee standaard tags eindigen, default = ]
    • result
      - het resultaat weergeven in HTML code in het result element, default = false

var myEditor = editor( input_element, options );
  • input_element

    Dit is het element waarin de gebruik zijn tekst bewerkt.

  • options

    De opties. Deze zijn gelijk aan de vorige opties, alleen result is niet mogelijk.

Alle elementen kunnen een string bevatten met het id van het element, of kunnen een HTML node zijn opgehaald door eenvan de DOM functies in JS.

Tags toevoegen

De tags toevoegen werkt ook simpel. Enig nadeel is dat je zelf moet uitzoeken wanneer je de functie wilt aanroepen. Je zal dus zelf een onclick event op een button moeten zetten en dan deze functie aanroepen.
Er zijn 2 functies: addNormalTag en addSpecialTag. Een normal tag is gewoon eentje die is opgebouwd uit [**tagnaam**]**tagvalue**[/**tagnaam**]. Een speciale tag is bijv. [url=**value**]**value**[/url].

addNormalTag

myEditor.addNormalTag( tag_name[, tag_result_name] );
  • tag_name

    Dit is de naam van de tag. Bijv. b of strong.

  • tag_result_name

    Dit is de naam die de tag heeft in het result veld. Voor bijv. de bold tag gebruik je in BB code [b] maar in HTML <strong> dit kun je aangeven bij tag_result_name

De tekst die geselecteerd is wordt tussen de 2 tags in gezet.

addSpecialTag

myEditor.addSpecialTag( tag_name, tag_input, tag_result );
  • tag_name

    De naam van de tag. Bijv. b of strong.

  • tag_input

    Een string die weergeeft hoe de tag eruit ziet in de editor. De ? wordt vervangen door de geselecteerde tekst. Bijv. [url=?]?[/url] maakt een url tag waarbij de ? 2x wordt vervangen door de geselecteerde tekst.

  • tag_result

    Een string die weergeeft hoe de tag eruit ziet in de result. Dus in HTML. Ook hierin wordt de ? vervangen door de geselecteerde tekst. Bijv. <a href="?">?</a> is de HTML versie van de anchor tag.

Met deze functie kun je je eigen tags maken.
